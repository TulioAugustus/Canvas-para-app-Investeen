# Investteen - Lean Canvas Interativo

Este repositório contém o **Lean Canvas Executivo e Interativo** do projeto **Investteen**, uma proposta de aplicativo gamificado de educação financeira focado em estudantes do Ensino Médio, auxiliado por Inteligência Artificial.

Este trabalho foi desenvolvido como parte de uma avaliação acadêmica universitária e validado através de pesquisas de campo com o público-alvo.

## Funcionalidades deste Canvas Interativo
Este não é um simples quadro estático. O arquivo HTML foi programado para funcionar como uma poderosa ferramenta de Pitch de Vendas:
**Modo Apresentação (Pitch):** Escurece a tela e destaca os 9 blocos na ordem lógica de um Pitch, navegável pelas setas do teclado.
**Filtro de Perspectiva Comercial:** Botões que adaptam instantaneamente os argumentos de venda para reuniões com Escolas Privadas (B2B) ou Secretarias Estaduais (B2G).
**Calculadora de Impacto:** Um simulador em tempo real onde o cliente digita o número de alunos e vê as estimativas de engajamento baseadas nos nossos dados de validação.
**Glassmorphism e Temas:** Design altamente moderno com botões para alternar entre *Dark Mode* e *Light Mode*.
**Exportação Otimizada:** Preparado nativamente para ser impresso ou salvo em PDF no formato A4 Paisagem, sem quebras e ocultando os controles automaticamente.

## Tecnologias Utilizadas
* **HTML5:** Estrutura semântica.
* **Tailwind CSS:** Estilização moderna e responsiva direto no arquivo.
* **JavaScript (Vanilla):** Lógica do modo de apresentação, calculadora e alternância de temas.
* **FontAwesome:** Biblioteca de ícones.
